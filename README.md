# website ini mengenai matematika untuk menghitung matematika dasar maupun bangun datar dan bangun ruang
website ini saya menggunakan jquery untuk digunakan dalam penghitungan serta menampilkan sebuah form dibawah foto, serta juga menggunakan html dan css, dan juga ada tambahan boostrap untuk desain tampilan website
halaman perhitungan tentang menghitung matematika dasar mengenai penjumlahan, pengurangan, perkalian, pembagian, pangkat, akar, persentase, yang dimana untuk menghitungnya harus mengklik gambar yang diinginkan
halaman bangun datar tetang untuk menghitung berbagai bangun datar mengenai persegi, persegi panjang, lingkaran, segitiga, trapesium, jajar genjang, yang dimana untuk memunculkan formnya harus klik gambar terlebih dahulu untuk memunculkan formnya
halaman bangun ruang tentang menghitung berbagai bangun ruang seperti kubus, balok, bola, silinder, kerucut, prisma, limas, tabung, untuk memunculkan formnya harus klik gambar terlebih dahulu untuk memunculkan formnya
